Website Research & Comparison

This section contrasts the components used in EcoMart against standard e-commerce practices and a reference website (e.g., "GenericShop").

Component / Feature EcoMart (Our Website) GenericShop (Investigated Website) Contrast / Reason
Navigation Bar Sticky, highlighting active page with JS. Static, no active state indication. Improves user orientation and navigation flow.
Product Display Dynamic rendering via JavaScript array. Hardcoded HTML items. Allows easier scalability and data management.
Forms Client-side validation with custom alerts. Basic HTML5 validation only. Enhances user experience by providing immediate, specific feedback.
Responsiveness CSS Grid and Flexbox for mobile-first design. Fixed width containers. Ensures accessibility across all device sizes.
Aesthetics Nature-themed color palette (Greens/Earth tones). Generic Blue/White corporate theme. Aligns visual identity with the sustainable mission.
